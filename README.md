# 使用

#### 安装启动：
安装依赖：
```
npm i
```
启动：
```
前端：
npm run serve
```   

#### 开始任务：
1.点开头部的项目设置，按需设置。<br />
2.设置结束后在首页选择要监听的产品及商铺名称。<br />
3.点击添加任务，确认无误后即开始监听。（可回到首页继续添加）<br />
4.可点开右上角的任务列表查看监听情况及结束任务<br />
